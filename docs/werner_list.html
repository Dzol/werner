Description

Werner Hett's list procedures and predicates in Erlang. These are the procedures and predicates from Werner Hett's 99 Prolog programs, but in Erlang, which was once built with a Prolog. The procedures herein take proper lists, i.e. lists terminated by the sentinal [].

Flat lists do NOT have elements which are lists themselves. Flatten a nested list structure: transform a list where elements may themselves be lists by building a list of their elements.

last/1

last(Rest::[term(), ...]) -> term()

The last element of a list. The last element of a list with only one element is that very element, otherwise the last element of a list is the last element of the rest of the list.

length/1

length(Rest::[term(), ...]) -> pos_integer()

The number of elements in a list. The list of just one element has length one, otherwise a list of more than one element has length one more than the length of the rest of the list.
